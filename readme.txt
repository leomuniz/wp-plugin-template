A WordPress plugin template for rapid plugin development.

== Description ==

WP Plugin Template provides a solid starting point for building WordPress plugins with modern tooling, including:

* PSR-4 autoloading via Composer
* Gutenberg block development with React and @wordpress/scripts
* WordPress Coding Standards (PHPCS)
* Lightweight unit testing with BrainMonkey
* Template rendering system for PHP views

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Go to Settings > WP Plugin Template to configure.

== Frequently Asked Questions ==

= How do I create a new plugin from this template? =

Copy the template directory, rename it, and follow the instructions in `CLAUDE.md` to replace all template references with your plugin name.
